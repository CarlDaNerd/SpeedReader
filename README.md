# Speed Reader

A browser-based RSVP (Rapid Serial Visual Presentation) speed reader. Paste in any text and read it word by word at speeds up to 1000 WPM.

## Features

- **RSVP display** — words flash one at a time with the optimal recognition point (ORP) pinned to the centre of the screen
- **Variable pacing** — long words and punctuation are held slightly longer to match natural reading rhythm
- **Context viewer** — pausing reveals a scrollable panel showing surrounding text with the current word highlighted in orange
- **Click to jump** — click any word in the context viewer to jump to that position and resume from there
- **Adjustable WPM** — slider, +/− stepper buttons, or keyboard arrows
- **Adjustable font size**
- **Dark / light mode**
- **Paste box** — hides during playback, reappears on pause or reset
- **Smart text parser** — correctly splits words at periods, em dashes, and run-together sentence boundaries

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / pause |
| `←` `→` | Step one word back / forward |
| `↑` `↓` | WPM up / down by 10 |

## Usage

1. Open `speed-reader.html` in any modern browser
2. Paste your text into the box at the bottom
3. Press **play**, hit `Space`, or click anywhere in the window 
4. Pause at any time to see the context viewer and jump to a different position

## No dependencies

Single self-contained HTML file. No internet connection required. Works in Chrome, Firefox, Safari, and Edge.
